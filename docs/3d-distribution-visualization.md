# 3D Distribution Visualization Spec

## 目的

確率分布を、3D 空間内で動く粒子と衝突体験として見せる。
統計の説明ではなく、分布が「発生する現場」を見せる。

## 基本原則

- 画面は 3D を基本とする
- 粒子は物理演算で動かす
- 分布は「箱に落ちた結果」や「通過回数」で表す
- 理論値とシミュレーション値の両方を表示する

## コア表現

### 正規分布

Galton board 形式を採用する。
多段のピンに当たりながら左右に分岐し、最終的な落下先の頻度が釣鐘型になる。

### 二項分布

各段で左右どちらかに進む離散的な分岐を使う。
段数を固定して、右に進んだ回数をそのまま結果にする。

### 超幾何分布

玉を戻さない抽選箱として表現する。
箱の中の色割合が、試行のたびに変化する様子を見せる。

### ポアソン分布

核分裂イベントの発生回数として扱う。
一定時間内に何回反応したかを数えると、Poisson の計測レイヤーになる。

### 指数分布

次の核分裂までの待ち時間として扱う。
核分裂イベント間の時間差をそのまま集計する。

### ガンマ分布

指数分布の待ち時間を k 回足し合わせる。
`k` 回目の核分裂までの累積時間を観測値にする。

### ベータ分布

MVP では外す。
ベイズ更新や成功率の事後分布として別枠で扱う方が本質に近い。

### 対数正規分布

核分裂回数や待ち時間の単純集計では出ない。
乗法的な増減を持つ別モデルが必要になる。

## reactor 流用メモ

`sidsenthilexe/reactor` のような核分裂シミュレーターを使う場合、Poisson / Exponential / Gamma は同じ本体に計測レイヤーを1枚乗せるだけで扱える。

- Poisson: 単位時間内の核分裂回数
- Exponential: 核分裂イベントの時間間隔
- Gamma: `k` 回目の核分裂までの累積時間

Lognormal はこの方式では作れないので、別モデルとして扱う。

## 3D での工夫

- ピンを平面に並べず、奥行きを持った層にする
- 層ごとに反発係数や傾きを少し変える
- 途中にリング状や円筒状の障害物を置く
- 落下後の粒子を床面に積層して地形化する
- カメラを固定せず、観察モードを切り替えられるようにする

## UI 方針

- 左ペイン: 分布選択
- 中央: 3D シミュレーション
- 右ペイン: パラメータと統計量
- 下部: ヒストグラムと理論曲線

## MVP

1. Galton board の 3D 版
2. 二項分布と正規分布の比較表示
3. 超幾何分布の抽選箱
4. ポアソン分布の 3D 着弾点マップ
5. 指数分布の待ち時間表現
6. パラメータ変更と再生成

## 拡張案

- 偏りのある Galton board
- 円筒状の Galton board
- 動的に変形するピン配置
- 粒子同士の相互作用あり版
- 分布の違いを色と音でも示すモード
- ベータ分布のベイズ更新モード

## 注意点

- 物理演算が重すぎると観察体験が崩れる
- 見た目を優先して統計的意味を壊しすぎない
- 最初は「わかりやすい分布」から入る
