# 3D Distribution Visualization Spec

## 目的

確率分布を、3D 空間内で動く粒子と衝突体験として見せる。
統計の説明ではなく、分布が「発生する現場」を見せる。

## 基本原則

- 画面は 3D を基本とする
- 粒子は物理演算で動かす
- 分布は「箱に落ちた結果」や「通過回数」で表す
- 理論値とシミュレーション値の両方を表示する

## コア表現

### 正規分布

Galton board 形式を採用する。
多段のピンに当たりながら左右に分岐し、最終的な落下先の頻度が釣鐘型になる。

### 二項分布

各段で左右どちらかに進む離散的な分岐を使う。
段数を固定して、右に進んだ回数をそのまま結果にする。

### 超幾何分布

玉を戻さない抽選箱として表現する。
箱の中の色割合が、試行のたびに変化する様子を見せる。

### ポアソン分布

3D 空間での着弾点やイベント発生位置を集計する。
検出器、宇宙線、落下点のような表現にすると見栄えが出る。

### 指数分布

次のイベントまでの待ち時間を、粒子の落下間隔や停止距離に変換する。

### ガンマ分布

複数回の待ち時間を足し合わせる舞台にする。
単発の指数分布よりも、少し厚みのある遷移になる。

### ベータ分布

MVP では外す。
ベイズ更新や成功率の事後分布として別枠で扱う方が本質に近い。

### 対数正規分布

毎回の増減が掛け算で進むような反発・増幅を与える。
普通の左右分岐よりも、右裾が長い見た目を作りやすい。

## 3D での工夫

- ピンを平面に並べず、奥行きを持った層にする
- 層ごとに反発係数や傾きを少し変える
- 途中にリング状や円筒状の障害物を置く
- 落下後の粒子を床面に積層して地形化する
- カメラを固定せず、観察モードを切り替えられるようにする

## UI 方針

- 左ペイン: 分布選択
- 中央: 3D シミュレーション
- 右ペイン: パラメータと統計量
- 下部: ヒストグラムと理論曲線

## MVP

1. Galton board の 3D 版
2. 二項分布と正規分布の比較表示
3. 超幾何分布の抽選箱
4. ポアソン分布の 3D 着弾点マップ
5. 指数分布の待ち時間表現
6. パラメータ変更と再生成

## 拡張案

- 偏りのある Galton board
- 円筒状の Galton board
- 動的に変形するピン配置
- 粒子同士の相互作用あり版
- 分布の違いを色と音でも示すモード
- ベータ分布のベイズ更新モード

## 注意点

- 物理演算が重すぎると観察体験が崩れる
- 見た目を優先して統計的意味を壊しすぎない
- 最初は「わかりやすい分布」から入る
